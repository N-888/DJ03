✨ О проекте
Это учебный, но выглядящий как современный мини-продукт сайт на Django: аккуратная структура проекта, база данных, админ-панель и стильный адаптивный интерфейс. Проект показывает полный цикл: модель → миграции → админка → вывод данных на сайте — плюс правильная работа со статикой без CDN.
________________________________________
✅ Что реализовано
🧩 Чистая структура Django-проекта
•	Отдельные приложения main и news для правильной архитектуры.
•	Маршруты разделены по приложениям и подключены через include().
•	Шаблоны разложены как “по учебнику”: templates/<app_name>/....
🗄️ База данных и модели
•	Создана модель NewsPost с полноценными полями:
o	название,
o	краткое описание,
o	полный текст,
o	дата публикации,
o	автор (связь с пользователем Django).
•	Настроена сортировка по дате: свежие новости отображаются первыми.
•	Удобное строковое представление записи (__str__) для админки.
🛠️ Профессиональная админ-панель
•	Модель подключена к админке.
•	В списке новостей отображаются ключевые поля (название, автор, дата).
•	Есть фильтры по автору и дате.
•	Есть поиск по тексту новости и по username автора.
📰 Полный вывод данных на сайте
•	Страница /news/ выводит все новости полностью, включая:
o	автора,
o	дату,
o	краткое описание,
o	полный текст.
•	Текст поддерживает переносы строк, чтобы читалось красиво.
________________________________________
🎨 Современный дизайн и UX
📱 Полная адаптивность
•	Интерфейс подстраивается под телефон/планшет/компьютер.
•	Меню превращается в “бургер” на маленьких экранах.
•	Сетка и карточки остаются удобными на любом размере.
💎 Визуально “как настоящий продукт”
•	Карточки с мягкими тенями и скруглениями.
•	“Воздух” в отступах и современная типографика.
•	Аккуратные бейджи и метаданные (автор/дата).
•	Приятная палитра и стиль, не похожий на “голый Bootstrap”.
🧭 Подсветка активного пункта меню
•	Текущая страница всегда выделяется в навигации.
•	Удобно ориентироваться и визуально понятно, где ты находишься.
🌓 Переключатель темы (Light/Dark)
•	В один клик можно переключать светлую и тёмную тему.
•	Выбор запоминается в браузере (localStorage).
⭐ Иконки в интерфейсе
•	Подключены Bootstrap Icons: меню и блоки стали визуально понятнее.
•	Иконки усиливают восприятие и делают UI более “дорогим”.
________________________________________
🚀 “По классике”: статика без CDN
•	Bootstrap подключён локально из папки static/.
•	Bootstrap Icons подключены локально, включая шрифты woff/woff2.
•	Проект работает без интернета и без внешних CDN — это важно для учебных заданий и “правильного” подхода.
